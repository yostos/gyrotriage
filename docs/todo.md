## スコアリング

- [x] ブレスコア（0〜100）の具体的な算出式を定義する
  - **解決**: RMS角速度ベース。`score = min(100, rms_velocity / 20.0 × 100)`。基準値20°/s。→ spec.md「スコア算出式」参照
- [x] 判定レベルの閾値を決定する
  - **解決**: STABLE(0-25, <5°/s), MILD(26-50, 5-10°/s), MODERATE(51-75, 10-15°/s), SEVERE(76-100, >15°/s)。→ spec.md「閾値テーブル」参照

## エラーハンドリング

- [x] モーションデータ非対応ファイルの扱いを決定する
  - **解決**: 非ゼロ終了コードでエラーメッセージを出力。NoMotionDataのみ特別フォーマット（撮影条件のヒント付き）。バッチ処理はツール外（シェル側）の責務（→ ADR-002）

## Gyroflow推奨パラメータ

- [x] ブレスコアからGyroflow推奨パラメータ（smoothness、crop）への対応ロジックを定義する
  - **解決**: 区分線形マッピング。スコア0-25→0.2, 26-50→0.3-0.5, 51-75→0.5-0.8, 76-100→0.8-1.5。crop推定: `1 + smoothness × RMS角速度 / FOV°`。→ spec.md「Gyroflow推奨パラメータ」参照
