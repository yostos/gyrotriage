# ADR-003: CSV出力の不採用

## ステータス

承認済み

## コンテキスト

初期構想では `--csv` オプションによるCSV出力を検討していた。他のツールや自作スクリプトとの連携用として、タイムスタンプごとの角変位量を含む詳細データを出力する想定であった。

## 決定

**`--csv` オプションは実装しない。**

## 根拠

- 確立されたユースケースがない。CSV出力を必要とする具体的なワークフローが特定されていない
- 本ツールの目的は人間の判断支援であり、機械間パイプライン連携ではない（ADR-002の単一ファイル原則とも整合）
- 必要になった時点で改めて検討すれば十分。YAGNI（You Aren't Gonna Need It）の原則に従う

## 影響

- `--csv` オプションは存在しない
- CSVフォーマットの定義や関連する依存クレート（csv等）は不要
- 将来的にユースケースが確立された場合は、新たなADRとして再検討する
